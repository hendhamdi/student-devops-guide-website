# 🌐 DevOps Guide – Site Vitrine

Bienvenue sur le site vitrine du projet **DevOps Guide for Students**.

Ce dépôt contient le code source du site web présentant l'ensemble de mes ressources pédagogiques dédiées au DevOps.

L'objectif est de proposer un point d'entrée simple pour découvrir les différents guides, documentations et ressources disponibles sur GitHub.

---

## 🎯 Objectif

Ce site permet de :

- Présenter les projets de documentation DevOps
- Faciliter l'accès aux différents dépôts GitHub
- Offrir une navigation plus agréable qu'un simple dépôt GitHub
- Centraliser les ressources destinées aux étudiants et débutants

---

## 📚 Ressources disponibles

Vous pourrez y retrouver notamment :

- Fondamentaux du DevOps
- Git & GitHub
- Docker
- CI/CD
- Kubernetes
- Monitoring (Prometheus & Grafana)
- Roadmaps
- Ressources d'apprentissage
- Documentation technique

---

## 🚀 Technologies utilisées

- React
- TypeScript
- Vite
- Tailwind CSS
- Lovable (design)
